# Places のユースケース

次に例を示します。

* ストアに入ったときにリアルタイム通知を送信する
* 自分の店舗と競合他社の店舗の足のトラフィックを分析する
* ビジネストラベラーオーディエンスの作成
* 該当する場合は、店舗でのエクスペリエンスを使用してユーザーをターゲティングします
